## Guía de inicio
## Requerimientos
1. NodeJs
2. Postgresql
## Instalación

1. Clonar el repositorio

2. Instalar las dependencias

```bash
npm install
```

3. Crear el archivo .env

```bash
cp .env.sample .env
```

4. Generar tablas de base de datos la base de datos

```bash
npm run prestart
```
5. Iniciar Servidor(Desarrollo)
```bash
npm run dev
```
6. Inciar Servidor(Producción)
```bash
npm run start
```
